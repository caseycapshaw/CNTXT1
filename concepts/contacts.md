# Concept — Contacts (People index)

**The index to the [`People/`](../People) folder.** Each person has their own note (the **single source of truth** for per-person detail — role, relationship, personal, interaction log, meeting history); this page is the **map + usage context** — *who to go to for what*. Click any name to open their note.

> **How to use this**
> - **Each person = one note**, filed as `People/Full Name.md`, wikilinked `[[Full Name]]`. Detail lives there, not here.
> - **Add a person:** run [[Add a person to the KB]] or copy `People/People TEMPLATE.md` → `People/<Full Name>.md`, fill it in, add a row here.
> - `status:` in each note — **active** (working with now) · **exec** (leadership/skip-level) · **dormant** (known, not engaged) · **departing** · **tbc** (name only).
